# Splice

A wheel of colored segments spins under a yellow notch. Tap when the target color crosses the notch.

## Play

- Watch the rotating wheel
- A color is named at the top
- Tap when that color is under the notch
- Speed climbs and segments multiply with score

## Build

```
flutter pub get
flutter run
flutter build apk --release
```
